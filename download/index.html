Artimus

What is Artimus?

Artimus was made to supplement an upcoming project called "Coffee-Engine", a competitor and stepping stone to everybody's favourite block based coding website.

In terms of it's original purpose it was supposed to be the asset creation portion of the engine, but eventually evolved into it's own standalone program. With it's own set of tools, settings, and features.

Free and Open-Source

Artimus is kept free and open thanks to users like you!

If you would like to contribute code you may on GitHub!
